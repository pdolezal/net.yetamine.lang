# net.yetamine.lang #

This repository provides a library with tiny extensions of the Java runtime libraries.


## Prerequisites ##

For building this project is needed:

* JDK 8 or newer.
* Maven 3.3 or newer.

For using the built library is needed:

* JRE 8 or newer.
